CVS Snapshots

This page contains snapshots of the CVS repository at various points in time. It is provided to disseminate pre-release and/or experimental versions of the software to specific users.

The software on this page is incomplete and untested; use it at your peril!

Otherwise, go here to get release versions of the software.

Date Packages Comments
23 Jun 2003
gazebo-src-23jun03.tar.gz
 Pre-release version of Gazebo with a working (hacky) Segway RMP model. This is compatable with the previous Player snapshot.
23 Jun 2003
player-src-23jun03.tar.gz
 This snapshot contains a working (but experimental) Player driver for the Segway RMP. PLEASE read this HowTo before trying it.
10 Jun 2003
gazebo-src-11jun03.tar.gz
player-src-11jun03.tar.gz
 This snapshots includes a pre-release version of Gazebo with the basic SegwayRMP model, and a corresponding version of Player containing the Gazebo drivers.
9 Jun 2003
gazebo-src-09jun03.tar.gz
player-src-09jun03.tar.gz
 This snapshots includes a pre-release version of Gazebo, and a corresponding version of Player containing the Gazebo drivers.
